Build your basket. Find the cheapest plan.

ShopSmart is a basket-optimization engine. Give it a shopping list and a price catalog, and it finds the cheapest way to shop — a single store, or a split across stores where each extra trip has to earn its place against a per-visit cost. The optimizer is real Python; this page mirrors its logic in the browser over the project's committed synthetic catalog.

01 — How it works

Three strategies, one honest trade-off.

The catalog is a table of offers — store sells product, in a pack of this size, for this price. From it the engine totals baskets, splits across stores, and weighs item savings against the cost of an extra trip.

🏪

Single store

Total each store’s basket (flagging items it doesn’t carry) and rank cheapest-first. One trip, whole list. best_single_store

✂️

Per-item split

Buy each item wherever it’s cheapest. A true lower bound on item cost — but it may send you to every store. per_item_cheapest

🎯

≤k-store plan

The real optimization: cheapest plan visiting at most k stores, with a fixed cost per trip. Solved exactly over store subsets. best_constrained_split

02 — Unit pricing

The cheaper sticker isn’t always the better deal.

Every pack is normalized to a base unit (kg / litre / each), so a larger pack with a higher price tag can still win on price-per-litre. From the committed catalog, Olive Oil:

StorePackPricePer litre
GreenBasket1 L€6.20€6.20 / L
MartMax750 ml€5.90€7.87 / L
ValuMart2 L€9.60€4.80 / L ✅

ValuMart’s bottle has the highest sticker price but the lowest price per litre, so best_unit_value("Olive Oil") picks it.

03 — The engine

Real, runnable, tested Python.

The optimizer is a small package with a pytest suite that cross-checks the ≤k-store optimizer against brute force. The worked example below is the actual output of demo.py (trip cost €1.00/store).

# pip install -r requirements.txt
# python -m pytest -q   ->  13 passed

from shopsmart import (
    demo_catalog, sample_shopping_list,
    best_single_store, per_item_cheapest, best_constrained_split,
)
cat, sl = demo_catalog(), sample_shopping_list()

best_single_store(cat, sl).total                                # 25.60 item
per_item_cheapest(cat, sl).item_cost                            # 23.55 (3 stores)
best_constrained_split(cat, sl, max_stores=2, visit_cost=1.0).total  # 25.90 all-in ✅
StrategyStoresItem costTripsAll-in
Best single store (GreenBasket)1€25.60€1.00€26.60
Per-item split3€23.55€3.00€26.55
Best ≤2-store (GreenBasket + MartMax)2€23.90€2.00€25.90 ✅

The per-item split has the lowest item cost but three trips push it to €26.55. The ≤2-store plan wins all-in at €25.90 — €0.70 (2.6%) under the best single store.
